FAQ

Frequently Asked Questions

Find answers to common questions about UIverse, components, usage, and contributions.

What is UIverse?

UIverse is a modern open-source UI component library built with HTML, CSS, and JavaScript for fast development.

Is UIverse free to use?

Yes, UIverse is completely free and open-source for personal and commercial projects.

Do I need any framework?

No frameworks required. UIverse works with pure HTML, CSS, and JavaScript.

Can I contribute components?

Yes! You can contribute by following the contribution guidelines in the docs section.

How do I start using UIverse?

Simply include the stylesheet and start using predefined classes like buttons, cards, and forms.
